# 📊 Univariate Analysis – Frequency Distribution & Interquartile Range (IQR)

## 📌 Project Overview

This project demonstrates **Univariate Analysis** using Python on the **Placement Dataset**. The objective is to understand the distribution of a single variable by calculating its frequency distribution and identifying potential outliers using the **Interquartile Range (IQR)** method.

This project is part of my Data Analytics learning journey and focuses on Exploratory Data Analysis (EDA) concepts.

---

## 🎯 Problem Statement

Before performing data analysis or building machine learning models, it is important to understand the characteristics of each variable.

This project aims to:

- Analyze the frequency distribution of the **ssc_p** (Secondary School Percentage) column.
- Calculate Relative Frequency and Cumulative Frequency.
- Compute Quartiles (Q1, Q2, Q3).
- Calculate the Interquartile Range (IQR).
- Detect potential outliers using the IQR method.
